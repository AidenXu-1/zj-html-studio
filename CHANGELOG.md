# 版本记录

ZJ HTML Studio 的重要变化记录在这里。

## 0.3.1 · 2026-07-17

- 修复 GitHub Release 来源证明与 Obsidian 自动审查器的兼容性；插件功能与 0.3.0 相同。

## 0.3.0 · 2026-07-17

- 新增“本地交互”模式，允许脚本渲染本地课件，同时继续阻止常见后台请求、外部子资源、Worker、嵌套页、表单、剪贴板和 `unsafe-eval`；页面导航与 WebRTC 不作绝对离线承诺。
- 安全只读检测到页面脚本时，在主画面、状态栏和诊断面板明确说明内容可能为空，并提供一键临时切换。
- Markdown 内嵌同步支持本地交互与脚本受限提示。
- 文件夹打开方式扩展为安全、本地交互和可信三态；旧版安全/可信规则继续有效。
- 刷新、权限升级和范围扩大改为后台候选验证；失败继续显示最后可用页面，成功保留当前子页、查询参数、锚点、滚动与源码阅读状态。
- HTML 使用已验证的不可变内存快照响应，统一校验 UTF-8、Range 和搜索桥；单文件上限 32 MiB，避免并发改写产生混合内容。
- 安全只读查找在有界预检后使用 Chromium 原生定位；本地交互与可信兼容改用可取消、分片执行的有界 DOM 定位，并显示“部分受限”，不对页面脚本创建的封闭区域伪报完整覆盖。源码查找继续显示精确计数。
- 补中文输入法、迟到结果、源码并发预算、会话/响应/文件句柄硬上限及键盘焦点保护。
- 补权限重置与初始加载交错、三标签 HTML 快照突发、Windows/旧式 Mac 换行源码坐标和服务繁忙诊断；快照在硬内存上限内有界排队，旧权限结果不能迟到接管。
- Markdown 嵌入候选页现在使用与正式画面相同的真实布局尺寸，接管时不重挂 iframe；分析与运行期诊断统一按资源身份去重，隐藏提醒不再重复计数。
- 大型 HTML 的脚本分析未完成时，标签页与 Markdown 嵌入都会解释空白风险并提供“本地交互打开”；嵌入诊断不再向界面暴露本机绝对路径。

## 0.2.0 · 2026-07-15

- 新增只读 HTML 源码查看，可与页面预览随时切换。
- 新增 50% 到 200% 页面缩放与一键回到 100%。
- 新增页面内查找、结果计数、上一处/下一处与 `Ctrl/Cmd+F`。
- 安全模式查找使用 nonce 限定的插件搜索桥，不放开用户脚本。
- 新增 Markdown `![[file.html]]` 嵌入，支持 `|760` 和 `|760x430` 尺寸。
- 嵌入会话按可见性延迟加载与回收，全局同时活上限为 8，并沿用资源范围、权限和自动刷新边界。
- 修复合法脚本或注释含 `</body>`、`</html>` 文本时，搜索桥可能插入用户源码中间的问题。
- 修复嵌入在加载中离屏时占用会话名额、离屏宽限期吞掉自动刷新，以及手动刷新未重新分析资源范围的问题。
- 修复范围确认重复弹出、弹窗随组件卸载未关闭，以及源码异步读取可能串到另一文件的问题。
- 修复离屏嵌入取消排队后异步等待不结束、长期滚动可能积累内存引用的问题。
- 修复并发刷新时旧加载可能释放新加载名额、造成会话覆盖或突破全局上限的竞态。
- Canvas、MHTML/MHT、移动端、HTML 编辑器和指定元素显示本轮不开启。

## 0.1.3 · 2026-07-15

第一个公开版本。

- 在 Obsidian 原生标签页预览本地 HTML 与 HTM 文件。
- 支持限定范围的本地资源、父目录路径、中文路径、查询参数、模块和媒体分段请求。
- 提供需要明确授权的“安全只读”与“可信兼容”模式。
- 提供依赖感知自动刷新、诊断、浏览器会话、源文件定位和通用全屏。
- 使用随机本地 Host 令牌和每会话资源范围隔离不同预览。
- 加入严格的生命周期、路径、并发和资源预算保护。
