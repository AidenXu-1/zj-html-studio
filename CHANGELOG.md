# 版本记录

ZJ HTML Studio 的重要变化记录在这里。

## 0.2.0 · 2026-07-15

- 新增只读 HTML 源码查看，可与页面预览随时切换。
- 新增 50% 到 200% 页面缩放与一键回到 100%。
- 新增页面内查找、结果计数、上一处/下一处与 `Ctrl/Cmd+F`。
- 安全模式查找使用 nonce 限定的插件搜索桥，不放开用户脚本。
- 新增 Markdown `![[file.html]]` 嵌入，支持 `|760` 和 `|760x430` 尺寸。
- 嵌入会话按可见性延迟加载与回收，全局同时活上限为 8，并沿用资源范围、权限和自动刷新边界。
- 修复合法脚本或注释含 `</body>`、`</html>` 文本时，搜索桥可能插入用户源码中间的问题。
- 修复嵌入在加载中离屏时占用会话名额、离屏宽限期吞掉自动刷新，以及手动刷新未重新分析资源范围的问题。
- 修复范围确认重复弹出、弹窗随组件卸载未关闭，以及源码异步读取可能串到另一文件的问题。
- 修复离屏嵌入取消排队后异步等待不结束、长期滚动可能积累内存引用的问题。
- 修复并发刷新时旧加载可能释放新加载名额、造成会话覆盖或突破全局上限的竞态。
- Canvas、MHTML/MHT、移动端、HTML 编辑器和指定元素显示本轮不开启。

## 0.1.3 · 2026-07-15

第一个公开版本。

- 在 Obsidian 原生标签页预览本地 HTML 与 HTM 文件。
- 支持限定范围的本地资源、父目录路径、中文路径、查询参数、模块和媒体分段请求。
- 提供需要明确授权的“安全只读”与“可信兼容”模式。
- 提供依赖感知自动刷新、诊断、浏览器会话、源文件定位和通用全屏。
- 使用随机本地 Host 令牌和每会话资源范围隔离不同预览。
- 加入严格的生命周期、路径、并发和资源预算保护。
